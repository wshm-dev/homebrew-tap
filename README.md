# wshm-dev/homebrew-tap

Homebrew tap for [wshm](https://github.com/wshm-dev/wshm) — AI-powered GitHub agent.

## Install

```bash
brew tap wshm-dev/tap
brew install wshm
```

## Update

```bash
brew update
brew upgrade wshm
```

## Uninstall

```bash
brew uninstall wshm
brew untap wshm-dev/tap
```

| Formula | Description |
|---------|-------------|
| `wshm` | AI-powered GitHub agent for repository maintenance |
